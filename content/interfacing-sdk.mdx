# Integrating SDKs with Dell Pro AI Studio

This guide shows you how to connect your applications to a locally running Dell-compatible inference service using the OpenAI-style SDKs. It includes setup, code samples in Python, JavaScript, .NET, cURL, streaming responses, and error handling.

---

### Prerequisites

* A running Dell Pro AI Studio Inference service at `http://localhost:8553/v1/openai`
* Models downloaded/cached on the local service or accessible via alias/model ID
* SDKs installed for your language (Python, JS, .NET)
* (Optional) Environment configured for authentication, if required

### Using the OpenAI SDK with Dell Pro AI Studio

<Tabs items={['Python', 'JavaScript', 'C#', 'cURL']}>
<Tabs.Tab>

```python
from openai import OpenAI

client = OpenAI(api_key="dpais", base_url="http://localhost:8553/v1/openai")

response = client.chat.completions.create(
    model="phi3.5",
    messages=[
        {
            "role": "system",
            "content": "You are a Dell B2B Marketing Assistant..."
        },
        {
            "role": "user",
            "content": "How can I create a LinkedIn campaign that highlights Dell Latitude laptops for hybrid workers?"
        }
    ]
)

print(response.choices[0].message.content)
```

### Streaming Responses

<Tabs items={['Python', 'JavaScript', 'C#', 'cURL']}>
<Tabs.Tab>

```python
for chunk in client.chat.completions.create(
    model="phi3.5",
    messages=[{"role": "user", "content": "Generate campaign taglines for hybrid laptop use"}],
    stream=True
):
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
```

</Tabs.Tab>
<Tabs.Tab>

```javascript
const stream = await openai.chat.completions.create({
  model: "phi3.5",
  messages: [{ role: "user", content: "Generate campaign taglines for hybrid laptop use" }],
  stream: true
});

for await (const part of stream) {
  const content = part.choices[0]?.delta?.content;
  if (content) process.stdout.write(content);
}
```

### Error Handling Examples

<Tabs items={['Python', 'JavaScript', 'C#', 'cURL']}>
<Tabs.Tab>

```python
from openai import OpenAI, APIError

client = OpenAI(api_key="dpais", base_url="http://localhost:8553/v1/openai")

try:
    response = client.chat.completions.create(
        model="phi3.5",
        messages=[{"role": "user", "content": "Give me a summary of Dell Latitude benefits"}]
    )
    print(response.choices[0].message.content)
except APIError as e:
    print(f"API request failed: {e}")
except Exception as e:
    print(f"Unexpected error: {e}")
```

</Tabs.Tab>
<Tabs.Tab>

```javascript
import OpenAI from "openai";

const openai = new OpenAI({
  baseURL: "http://localhost:8553/v1/openai",
  apiKey: "dpais",
});

async function run() {
  try {
    const completion = await openai.chat.completions.create({
      model: "phi3.5",
      messages: [{ role: "user", content: "Give me a summary of Dell Latitude benefits" }],
    });
    console.log(completion.choices[0].message.content);
  } catch (err) {
    console.error("Request failed:", err.message || err);
  }
}

run();
```
