# Dell Pro AI Studio

Dell Pro AI Studio (`dpais`) is an AI inference platform designed to optimize deployment of on-device AI models and provide manageability for locally-run AI applications.

- Seamless integration with existing SDKs, via our CLI, or OAI REST API endpoints.
- CLI support & Enterprise management for fast initialization and model lifecycle control.

You can install Dell Pro AI Studio using the CLI or download the MSI installer manually from [dell.hf.co](https://dell.hf.co).
